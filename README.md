# WAIC 2026 议程导航器

> 2026 世界人工智能大会（WAIC）175 场论坛的交互式可视化导航工具

暗黑科技风单页 Web 应用，帮助你从 175 场论坛中快速定位感兴趣的活动——时间、地点、赛道一目了然。

## 在线访问

**GitHub Pages**: https://daka-agent.github.io/WAIC2026/

## 功能特性

### 时间轴视图
- 按 4 天（7月17-20日）分组展示，时间标尺定位
- 每场活动卡片左侧有赛道专属色条，一眼区分领域
- 点击卡片弹出详情弹窗，查看完整信息

### 交互式地图
- 基于 Leaflet 的上海真实地图（深色底图）
- 7 大场馆 + 8 个分散场地标注为霓虹图钉
- 图钉大小反映该场馆活动数量
- 点击图钉弹出该场馆所有活动列表

### 赛道分类筛选
- 13 条赛道各有专属霓虹色
- 导航栏可多选筛选，实时联动所有视图
- 赛道视图展示每条赛道的核心方向和适合行业

### 全文搜索
- 实时搜索论坛名称、赛道、场馆
- 搜索 + 赛道筛选 + 日期筛选三重联动

### 收藏日程
- 点击星标收藏感兴趣的活动
- "我的日程"视图按天展示已收藏活动
- localStorage 本地持久化，关闭页面不丢失

## 数据来源

基于 `WAIC 2026世界人工智能大会日程表（分类导航版）.xlsx` 提取，包含：

| 维度 | 数量 |
|------|------|
| 论坛总数 | 175 场 |
| 会议天数 | 4 天（7月17-20日） |
| 赛道分类 | 13 条 |
| 会议场馆 | 7 个主场馆 + 8 个分散场地 |

## 技术栈

- **Tailwind CSS** (CDN) — 快速样式开发
- **Leaflet** (CDN) — 交互式地图
- **Orbitron + Inter** — 科技感字体组合
- **Vanilla JS** — 无框架依赖，单文件部署
- **localStorage** — 本地收藏持久化

## 本地使用

### 方式一：直接打开
下载 `index.html`，双击即可在浏览器中打开。

### 方式二：本地服务器
```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve
```
然后访问 http://localhost:8080

## 设计风格

- **配色**：深蓝黑背景 (#060912) + 霓虹荧光渐变（青/紫/粉）
- **质感**：玻璃拟态卡片 + 浮动光晕 + 网格底纹
- **字体**：Orbitron（标题/科技数字）+ Inter（正文）
- **交互**：卡片悬浮动画 + 弹窗过渡 + 筛选联动
